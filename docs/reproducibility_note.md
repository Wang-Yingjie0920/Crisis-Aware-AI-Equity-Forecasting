# Reproducibility Note

This repository is a research portfolio version of the project.

The full local project includes a longer day-by-day empirical pipeline from data construction to model training, uncertainty calibration, backtesting, mechanism interpretation, and final report packaging.

For GitHub presentation purposes, this repository provides selected reports, figures, tables, and documentation.

The uploaded files are intended to demonstrate the research design, empirical evidence, and final reporting structure. Raw intermediate files and large local outputs are not included in this public repository.
