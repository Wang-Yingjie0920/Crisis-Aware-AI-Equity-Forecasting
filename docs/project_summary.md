# Project Summary

This project develops a long-history crisis-aware AI forecasting framework for U.S. equity indices.

The empirical system combines:

1. Long-history S&P 500 and Nasdaq-100 data
2. Macro-financial feature engineering
3. Crisis-regime and vulnerability indicators
4. Multi-horizon forecasting targets
5. Baseline, tree-based, boosted, ensemble, routing, and stacking models
6. Conformal uncertainty calibration
7. Backtesting and robustness checks
8. Mechanism interpretation and hypothesis testing
9. Final report, reference, figure, and table packaging

The final report should be interpreted as a research-oriented empirical finance and financial machine learning project, not as financial advice or a guaranteed trading system.
